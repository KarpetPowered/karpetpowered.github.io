Configurable

Karpet is easily configurable via its config file(soon)

Built with Minestom

Minestom is a open-source java framework for building minecraft servers, it comes with support for extensions, and is multi-threaded unlike the default minecraft server. This allows Karpet to perform better than the default minecraft server.
